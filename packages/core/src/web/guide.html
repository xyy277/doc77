Sidebar

The left panel centralizes project switching, file navigation, search, and other core operations.

💼
 Switch Project Click the top button to switch between registered project directories. Each project manages its own file structure.
⭐
 Favorites Right-click a file to add it to favorites for quick access to frequently used documents. Click to open without navigating directories.
🕐
 Recent Files Automatically tracks recently opened files for quick backtracking. No need to browse the tree every time.
📂
 File Tree Browse project folder structure. Click directories to expand or collapse, click files to preview content instantly.
🔍
 Filter Files Type keywords above the file tree to quickly locate target files. Supports fuzzy matching.
🔎
 Global Search Search all file content within the project (file names + full text). Press Enter to trigger, results are highlighted.
◀
 Collapse Sidebar Collapse the sidebar into a narrow strip (logo only) for more document reading space.
🌙
 Theme Toggle between dark and light modes to match your reading environment and protect your eyes.
⚙
 Settings Configure AI Token and model, network bind address, access password, editor, and other global options.
▶

Reading Toolbar

The top toolbar provides rich reading aids for more efficient document browsing.

🔍
 Find in Document Search for keywords in the current document, supports ↑ ↓ to jump between matches. Shortcut Ctrl+F
▶
 Auto Scroll Automatically scrolls down for hands-free reading with four speed settings (Slow / Medium / Fast / Instant).
🔊
 TTS Read Aloud Reads documents aloud with five speed settings (0.5x — 2x). Listen when your eyes are tired.
✨
 AI Summary Generate a document summary with one click, with read-aloud support. Quickly understand the core content of long documents. Requires AI Token configuration.
▶
 Run Code Open .js or .py files to run directly in the browser Sandbox or Pyodide (WASM).
📝
 Open in Editor Open the current file in VS Code or your system default editor for editing with one click.
📂
 Show in Folder Locate and highlight the current file in the system file manager (Finder / File Explorer).
📑

Right Panel

The right panel integrates the AI assistant, document outline, and approval queue — the intelligent core of Doc77.

📑
 Document Outline Automatically generates a document structure navigation from Markdown headings. Click headings to quickly jump to the corresponding section.
💬
 AI Assistant A conversational AI that can answer questions about the current document, request summaries, and plan batch file operations. Built on the MCP protocol, the AI can safely read and write files.
📋
 Approval Queue MCP write operations (create/modify/delete files) enter the approval queue by default and are executed only after confirmation. Supports CLI and Web dual-channel approval with transaction rollback.
📊

Dashboard

The home dashboard provides project overview, statistics, and quick action shortcuts.

🕐
 Recent Browsing View recently opened projects and files, jump with one click. Supports project-level and file-level history.
📊
 Statistics Displays key metrics at a glance: total projects, favorites count, last active time, and more.
⭐
 Favorite Projects Starred favorite projects are showcased together, with quick unfollow support.
📂
 Project List All projects displayed in grid or list view. Supports search, sorting (recent / name / created date), and view switching.
➕
 Register Project Two ways to add a project: manually enter a path, or auto-scan directories to discover registerable project directories.
