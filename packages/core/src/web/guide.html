侧边栏

左侧面板集中了项目切换、文件导航、搜索等核心操作。

💼
 项目切换 点击顶部按钮切换已注册的项目目录，每个项目独立管理其文件结构。
⭐
 收藏夹 右键文件选择收藏，常用文档快速访问。点击即打开，无需重复翻目录。
🕐
 近期文件 自动记录最近打开的文件，方便快速回溯。不必每次都翻目录树查找。
📂
 目录树 浏览项目文件夹结构，点击目录展开或收起，点击文件即时预览内容。
🔍
 文件过滤 在目录树上方输入关键字，快速定位目标文件，支持模糊匹配。
🔎
 全局搜索 搜索项目内所有文件内容（文件名 + 全文），回车触发，结果高亮显示。
◀
 折叠侧栏 收起侧栏为窄条（仅显示 Logo），获得更大的文档阅读空间。
🌙
 主题切换 一键切换深色或浅色模式，匹配你的阅读环境，保护眼睛。
⚙
 设置 配置 AI Token 与模型、网络绑定地址、访问密码、编辑器等全局选项。
▶

阅读工具栏

顶部工具栏提供了丰富的阅读辅助功能，让文档浏览更高效。

🔍
 文档内搜索 在当前文档中搜索关键词，支持 ↑ ↓ 跳转匹配项。快捷键 Ctrl+F
▶
 自动滚动 自动向下滚屏阅读，四档速度可调（慢/中/快/极速），解放双手。
🔊
 TTS 朗读 语音朗读文档，五档语速调节（0.5x — 2x）。眼睛累了就用听的。
✨
 AI 总结 一键生成文档摘要，支持朗读摘要。快速了解长篇文档核心内容。需配置 AI Token。
▶
 运行代码 打开 .js 或 .py 文件时，可在浏览器 Sandbox 或 Pyodide (WASM) 中直接执行。
📝
 编辑器打开 一键在 VS Code 或系统默认编辑器中打开当前文件进行编辑修改。
📂
 文件夹中显示 在系统文件管理器（Finder / 资源管理器）中定位并高亮当前文件位置。
📑

右侧面板

右侧面板集成了 AI 助手、文档大纲和审批队列，是 Doc77 的智能核心。

📑
 文档大纲 根据 Markdown 标题自动生成文档结构导航，点击标题快速跳转至对应章节。
💬
 AI 助手 对话式 AI，可针对当前文档提问、请求总结、规划批量文件操作。基于 MCP 协议集成，AI 可安全读写文件。
📋
 审批队列 MCP 写操作（创建/修改/删除文件）默认进入审批队列，确认后才执行。支持 CLI 与 Web 双通道审批，事务回滚。
📊

仪表盘

首页仪表盘提供项目总览、统计数据和快捷操作入口。

🕐
 最近浏览 查看最近打开的项目和文件，一键跳转。支持项目粒度和文件粒度的历史记录。
📊
 统计概览 显示项目总数、收藏数、最近活跃时间等关键指标，一目了然。
⭐
 收藏项目 星标收藏的项目集中展示，支持快速取消收藏。
📂
 项目列表 全部项目以网格或列表视图展示。支持搜索、排序（最近打开/名称/创建日期）、视图切换。
➕
 注册项目 支持两种方式添加项目：手动输入路径，或自动扫描目录发现可注册的项目目录。
📝
