左侧栏 侧边栏

💼项目切换点击顶部按钮切换已注册的项目目录，每个项目独立管理
⭐收藏夹右键文件选择收藏，常用文档快速访问，点击即打开
🕐近期文件自动记录最近打开的文件，不必重复翻目录树
📁目录树浏览项目文件夹结构，点击目录展开/收起，点击文件即时预览
🔍文件过滤在目录树上方输入关键字，快速定位目标文件
🔎全局搜索搜索项目内所有文件内容（文件名 + 全文），回车触发
◀折叠侧栏收起侧栏为窄条（仅显示 Logo），获得更大阅读空间
🌙主题切换一键切换深色 / 浅色模式，保护你的眼睛
⚙设置配置 AI Token 与模型、网络绑定地址、访问密码、编辑器等全局选项

顶部工具栏 阅读工具

🔍文档内搜索在当前文档中搜索关键词，支持 ↑↓ 跳转匹配项。快捷键 Ctrl+F
▶自动滚动自动向下阅读滚动，四档速度可调（慢 / 中 / 快 / 极速），解放双手
🔊TTS 朗读语音朗读文档，五档语速调节（0.5x - 2x），眼睛累了就用听的
✨AI 总结一键生成文档摘要，支持朗读摘要，快速了解长篇文档核心内容。需配置 AI Token
▶运行代码打开 .js / .py 文件时，可直接在浏览器 Sandbox / Pyodide 中执行
📝编辑器打开一键在 VS Code 或系统默认编辑器中打开当前文件进行编辑
📂文件夹中显示在系统文件管理器（Finder / 资源管理器）中定位并高亮当前文件

右侧面板 AI 与审批

📑文档大纲根据 Markdown 标题自动生成文档结构导航，点击标题快速跳转
💬AI 助手对话式 AI，可针对当前文档提问、请求总结、规划批量文件操作。MCP 协议集成，AI 可安全读写文件（需审批）
📋审批队列MCP 写操作（创建/修改/删除文件）默认入队等待审批。你确认后才执行，支持 CLI 与 Web 双通道审批，事务回滚保障安全
